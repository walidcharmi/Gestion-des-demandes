# 🏢 Gestion des demandes des parties prenantes - ENET'Com

Application web développée avec Google Apps Script permettant la gestion des demandes des étudiants, enseignants et administratifs de l’ENET’Com.

---

## 🚀 Objectif du projet

Centraliser et digitaliser les demandes des utilisateurs de l’ENET’Com via un système en ligne simple et structuré.

---

## ✨ Fonctionnalités principales

- 📋 Soumission de demandes en ligne (étudiants / enseignants / administratifs)
- 📎 Upload de fichiers (PDF, Word selon le type de demande)
- 🧾 Gestion des demandes de matériels
- 🎓 Gestion des activités estudiantines
- 📄 Dépôt de documents (PFE et thèses)
- 📧 Traitement et envoi automatique via Google Apps Script
- 🔍 Validation des formulaires côté client

---

## 🛠️ Technologies utilisées

- Google Apps Script
- HTML5 / CSS3
- JavaScript (front-end)
- FileReader API (gestion des fichiers)
- Google Drive / Google Sheets (backend)

---

## 📁 Structure du projet

- `Index.html` → Interface utilisateur principale
- `Code.gs` → Logique serveur (Google Apps Script)
- Upload de fichiers intégré (PDF / DOCX)

---

## ⚙️ Fonctionnalités techniques

- Gestion dynamique des formulaires selon le type de demande
- Validation des fichiers uploadés
- Lecture des fichiers via Base64
- Soumission vers Google Apps Script (`google.script.run`)

---

## 🌐 Utilisation

1. Ouvrir l’application web
2. Remplir le formulaire selon le type de demande
3. Joindre les fichiers nécessaires
4. Soumettre la demande
